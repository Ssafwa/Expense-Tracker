Expense Tracker Web Application
Overview
The Expense Tracker is a personal finance management tool designed to help users monitor their income and expenses in real-time. The application provides a clear dashboard to visualize spending patterns, categorize transactions, and maintain a historical record of all financial activity.

Key Features
Real-time Dashboard: Instantly view total income and monthly expenses.

Transaction Management: Easily add, categorize, and track income and expense entries.

Interactive Modals: User-friendly pop-up forms for quick data entry.

Visual Overview: Monthly charts and category-based spending breakdowns.

Responsive UI: Designed for a clean, professional user experience.

Languages & Technologies Used
HTML5: Structured the semantic web layout.

CSS3: Styling, responsive design, and animations (including FontAwesome for icons).

JavaScript (ES6+): Core logic, DOM manipulation, transaction processing, and modal management.
